In this assignment I learned to code in C#. I learned a lot of the basics, included using namespaces, naming conventions, and using the debugger. I found myself confused at first, but then I fell in love with the language. It seems really fantastic and I am excited to learn more and use it more often.

  1. The first thing I did was to create a new branch on my github website. I did this using the git commands:
    git branch -b hw3
    git checkout hw3
    And then I began working on that branch, pushing changes, creating files, all without affecting my main branch. Source control is truly amazing, and I wil be using it in all future projects, for almost any class I have.
  2. Secondly I opened up the Java program Calculator and looked it over. There were 4 files and a fair bit of code, I tried to understand how it worked, ran the calculator a couple of times, and got a good feeling for the program.
  3. Then I began working in Visual Studio 2017.

    I've used Visual Studio in the past, but never too much. I found myself a little intimidated, as it's a mature program with a lot of menus and options. However, I was able to get a hold of the basics fairly quickly and found myself a good workflow.
  4. Up next came the actual translation of the code from Java to C#.
