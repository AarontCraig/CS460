Game Details

SoE is a throwback to the 90's era of RPG's. Back then story and character reigned supreme, along with unique and engaging combat. Think Final Fantasy 3, Chrono Trigger, and Secret of Mana. Expect an epic storyline, centered on characters, brand new gameplay, and a leveling system like you've never experienced. Look forward to:
